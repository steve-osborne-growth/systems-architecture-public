# Component: Compliance Tracking

## Purpose

The compliance tracking dashboard gives every stakeholder a real-time view of where each customer stands in the EMS migration.

---

## What It Shows

- Assigned user counts versus contracted license counts per customer per product
- Current compliance status: compliant, in seller-led period, in countdown, or fully migrated
- Progress metrics showing how many customers have completed migration across each wave
- Historical tracking of status changes over time

---

## Design Notes

This dashboard is built in Domo and sits on top of the normalized data pipeline. It was built to serve the audit requirement: leadership needs to see at any point how far the organization has progressed through the transition. Without this visibility, the migration would be impossible to manage at scale.
